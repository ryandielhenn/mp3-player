# Mp3 Player

## Usage
```
# Compile the project
make compile

# Compile and run
make run

# Clean up compiled files
make clean

# Install to ~/.local/bin/mp3
make install

# Run program
mp3

# Uninstall from ~/.local/bin
make uninstall
```
