# Mp3 Player

## Usage
```
# Compile the project
make compile

# Compile and run
make run

# Clean up compiled files
make clean
```

## Screenshots

### Searching for music in a directory
Searches the given directory and all subdirectories for Mp3 files.
Defaults to `$HOME/Music` (e.g.) `/home/ryan/Music`
